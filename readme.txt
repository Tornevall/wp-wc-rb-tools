Independent utility plugin for WooCommerce with Resurs Bank integration.

== Description ==

Tornevalls Toolbox for Resurs Bank is an independent third-party utility plugin for WooCommerce sites using Resurs Bank as a payment provider.

This plugin is not officially affiliated with, endorsed by, maintained by, or supported by Resurs Bank.

It is built to support inspection, troubleshooting, status checks, and utility functions alongside Resurs Bank payment integrations for WooCommerce.

== Installation ==

1. Upload to `/wp-content/plugins/tornevalls-tools-for-resurs-bank-payments/`
2. Activate via the Plugins screen in WordPress
3. Go to WooCommerce > Tornevalls Toolbox for Resurs Bank
4. Make sure WooCommerce and Resurs Bank Payments for WooCommerce are installed and activated

== Frequently Asked Questions ==

= Do I need the Resurs Bank Payments for WooCommerce plugin? =

Yes. This plugin is designed to work alongside Resurs Bank Payments for WooCommerce. Without it, the plugin will have limited or no practical functionality.

= Is this supported by Resurs Bank? =

No. This is an independent third-party utility plugin.

= Can I use this in production? =

This is a utility/status tool. Use it at your own discretion and test in your own environment before production use.
